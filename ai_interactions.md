## Agent Workflow (SF8)

> Document your experience using an AI agent (e.g., Cursor Agent, Claude, Copilot) to make multi-step changes autonomously.

**AI tool / mode used:** Claude Code (Opus 4.8) running in agent mode — the agent read the
files, made edits, and ran commands autonomously with my approval.

**What task did you give the agent?**

- Investigate the planted game glitches and fix them (inverted hints, non-progressive
  difficulty ranges, the off-by-one attempts counter, and the New Game button not fully
  resetting state).
- Audit `app.py` for business logic mixed into the Streamlit UI code.
- Refactor: move the `check_guess` function out of `app.py` into `logic_utils.py` and update
  the import.
- Generate pytest cases that specifically target the bugs that were fixed.

**What did the agent do?**

- Fixed the inverted hints, difficulty ranges, attempts off-by-one, hardcoded guess prompt,
  and New Game reset in `app.py` (each change marked with a `# Bug fix:` comment).
- Moved `check_guess` into `logic_utils.py` and added `from logic_utils import check_guess`
  to `app.py`.
- Rewrote `tests/test_game_logic.py` (the original tests compared the `(outcome, message)`
  tuple to a bare string and were failing) and added regression tests for the inverted-hint
  bug plus the string-comparison fallback path.
- Ran `python -m pytest tests/test_game_logic.py -v` to confirm all 6 tests pass, and ran a
  syntax/import check on `app.py` and `logic_utils.py`.

**What did you have to verify or fix manually?**

- The agent first tried a broader refactor (extracting `get_attempt_limit` and a `step_game`
  orchestrator); I declined that and narrowed it to just moving `check_guess`.
- The agent flagged a duplicate test directory (`tests/` already existed vs. the requested
  `test/`) and asked where to put the tests instead of creating a confusing second folder.
- Verified the running Streamlit game behaved correctly after the fixes (hints point the
  right direction, New Game fully resets, difficulty ranges grow Easy < Normal < Hard).
